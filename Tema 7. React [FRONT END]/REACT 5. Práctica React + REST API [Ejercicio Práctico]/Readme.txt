# Weather App

Aplicación de clima simple construida con React y Material-UI que consume la API de WeatherAPI.com.

## Características

- Búsqueda del clima por ciudad
- Visualización de temperatura actual, condiciones climáticas e icono representativo
- Manejo de errores y validación de formularios
- Interfaz responsive y moderna con Material-UI
- Indicador de carga durante las peticiones a la API

## Tecnologías utilizadas

- React
- Material-UI (MUI)
- WeatherAPI.com

## Configuración

1. Clona el repositorio
2. Instala las dependencias con `npm install`
3. Crea un archivo `.env` en la raíz del proyecto y agrega tu API key:
   ```
   VITE_API_KEY=tu_api_key_aqui
   ```
4. Inicia la aplicación con `npm run dev`

## Uso

1. Ingresa el nombre de una ciudad en el campo de búsqueda
2. Haz clic en el botón "Buscar"
3. Visualiza la información del clima actual
