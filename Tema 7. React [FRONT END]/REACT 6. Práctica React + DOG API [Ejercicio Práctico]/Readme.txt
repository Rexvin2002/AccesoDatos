# Dog Image App

Una aplicación simple que muestra imágenes aleatorias de perros según la raza seleccionada, utilizando la API pública de Dog CEO.

## Características

- Listado de razas de perros disponibles
- Selección de raza mediante un dropdown
- Visualización de imágenes aleatorias de la raza seleccionada
- Manejo de estados de carga y errores
- Diseño responsive con Material-UI

## Tecnologías utilizadas

- React
- Material-UI (MUI) para los componentes de UI
- Fetch API para las llamadas HTTP
- React Hooks (useState, useEffect)

## Instalación

1. Clona este repositorio
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia la aplicación:
   ```bash
   npm start
   ```

## Uso

1. Selecciona una raza de perro del dropdown
2. Haz clic en "Buscar Imagen"
3. La aplicación mostrará una imagen aleatoria de un perro de la raza seleccionada

## API utilizada

Esta aplicación utiliza la API pública de [Dog CEO](https://dog.ceo/dog-api/).
