# README - ConsultasXPath

## Descripción
Programa Java que realiza consultas XPath sobre un archivo XML de catálogo de libros (BOOKS.xml). Demuestra diferentes tipos de consultas XPath para extraer información específica del documento XML.

## Funcionalidades
- Consulta y muestra los títulos de todos los libros
- Busca el autor de un libro específico por ID
- Filtra libros por precio (mayores a 30)
- Lista fechas de publicación
- Obtiene información del primer libro del catálogo

## Requisitos
- Java 8 o superior
- Archivo BOOKS.xml en el mismo directorio

## Uso
1. Compilar:
```
javac ConsultasXPath.java
```

2. Ejecutar:
```
java ConsultasXPath
```

## Consultas implementadas
1. Todos los títulos de libros
2. Autor del libro con ID 'bk102'
3. Libros con precio > 30
4. Fechas de publicación
5. Primer libro del catálogo

## Salida
El programa muestra en consola los resultados de cada consulta con formato claro.
