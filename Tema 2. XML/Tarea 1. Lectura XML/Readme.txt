# README - XMLReader

## Descripción
Programa Java para visualizar archivos XML con formato indentado. Muestra la estructura completa del documento XML con sus atributos y contenido.

## Características
- Muestra la estructura jerárquica del XML con indentación
- Incluye atributos de cada nodo
- Omite espacios en blanco y comentarios
- Formatea el texto para mejor legibilidad

## Uso
Compilar y ejecutar con el archivo XML como argumento:

```
javac XMLReader.java
java XMLReader archivo.xml
```

## Requisitos
- Java 8 o superior
- Archivos XML bien formados
