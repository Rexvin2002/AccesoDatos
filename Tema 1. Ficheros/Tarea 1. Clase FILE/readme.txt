# README - FileClass

## Descripción
Programa Java para explorar archivos y directorios. Muestra información detallada y calcula tamaños recursivamente.

## Funcionalidades
- Muestra detalles de archivos/directorios
- Lista contenidos de directorios
- Calcula tamaño total (incluyendo subdirectorios)
- Modo interactivo o por argumento

## Uso
1. Compilar: `javac FileClass.java`
2. Ejecutar: 
   - Interactivo: `java FileClass`
   - Con ruta: `java FileClass /ruta/ejemplo`

**Durante ejecución:**
- Ingrese ruta a examinar
- Presione `1` para salir
