Vamos a realizar un programa para realizar 3 funciones (se puede utilizar un menú para actuar):

1º CREAR. Solicita un nombre de fichero, se crea un fichero .txt y se pide al usuario escribir el contenido que se introducirá al fichero.

2º AÑADIR FINAL. Solicita un nombre de fichero al usuario, se abre el fichero, y se pide al usuario texto que se incluirá al final del fichero.

3º AÑADIR PRINCIPIO. Solicita un nombre de fichero al usuario, se abre el fichero, y se pide al usuario texto que se incluirá al comienzo del fichero.

Utilizaremos las clases File, FileWriter, BufferedWriter, BufferedReader, etc.
